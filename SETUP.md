# Сніжинка Коха

Програма для генерації та візуалізації фракталу "Сніжинка Коха" різних рівнів рекурсії.

## Встановлення залежностей

### Варіант 1: Автоматичне встановлення
Просто запустіть програму - залежності встановляться автоматично:
```bash
python task_02_v03_1.py
```

### Варіант 2: Ручне встановлення
```bash
pip install -r requirements.txt
```

## Запуск

```bash
# З активованим віртуальним окруженням
python task_02_v03_1.py

# Через скрипт-обгортку (перевіряє залежності)
python run.py
```

## Вимоги

- Python 3.7+
- matplotlib >= 3.5.0

## Використання

1. Запустіть програму
2. Введіть рівень рекурсії (0-7)
3. Програма згенерує зображення сніжинки та збереже його в папку `task_02_koch_snowflake/`
4. Натисніть Enter без введення, щоб вийти

## Структура папок

```
task_02_koch_snowflake/
├── koch_snowflake_level_0.png
├── koch_snowflake_level_1.png
├── koch_snowflake_level_2.png
└── ...
```

## Файли

- `task_02_v03_1.py` - Основна програма з автоматичною обробкою залежностей
- `run_task2.py` - Скрипт-обгортка для перевірки залежностей
- `requirements.txt` - Список залежностей

## Приклади запуску

```bash
# Рівень 0
Введіть рівень рекурсії (0 до 7): 0

# Рівень 3
Введіть рівень рекурсії (0 до 7): 3

# Вихід
Введіть рівень рекурсії (0 до 7): [натисніть Enter]
```

## Результати

Програма створює файли виду `koch_snowflake_level_N.png` у папці `task_02_koch_snowflake/` де N - рівень рекурсії.

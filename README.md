# goit-algo-hw-03
Homework for GoIT. Basic Algorithms. Theme 3

## Завдання

### Завдання 1: Копіювання файлів за розширеннями
Реалізувати програму для сортування та копіювання файлів з вихідної папки в цільові папки за їх розширеннями.

**Принцип роботи:**
- Рекурсивно сканує всі підпапки вихідної директорії
- Групує файли за розширеннями (розширення = назва папки для копіювання)
- Копіює файли в відповідні папки за розширенням
- Невідомі розширення копіюються в папку "other"

**Використання:**
```bash
python task_01_v01.py <path_to_source_directory>
```

**Файли:** `task_01_v01.py`

### Завдання 2: Сніжинка Коха
Побудова фракталу "Сніжинка Коха" за допомогою рекурсії з різними рівнями деталізації та візуалізацією за допомогою matplotlib.

**Принцип роботи:**
- Користувач вибирає режим виводу: тільки файл або файл + графічне вікно
- Користувач вводить рівень рекурсії (0-7)
- Програма розраховує координати точок фракталу використовуючи рекурсивну функцію
- На кожному рівні відрізок ділиться на 3 частини з додаванням "зубця" під кутом 60°
- Графік малюється matplotlib і зберігається в папку `task_02_koch_snowflake/`
- За вибраним режимом також може показані в графічному вікні
- Залежності (matplotlib) встановлюються автоматично при першому запуску
- Можна генерувати кілька сніжинок різних рівнів в одному сеансі

**Файли:** `task_02_v03_1.py`, `SETUP.md`

## Запуск Завдання 2

```bash
# Активувати віртуальне окруження
source .venv/bin/activate

# Запустити програму
python task_02_v03_1.py

# 1. Оберіть режим виводу (0 або 1)
# 2. Введіть рівень рекурсії (0-7)
# 3. Графіки будуть збережені в папку task_02_koch_snowflake/
# 4. Повторіть для інших рівнів або натисніть Enter для виходу
```

**Або через скрипт-обгортку (перевіряє та встановлює залежності):**

```bash
python run_task2.py
```

**Режими виводу:**
- **0** — тільки збереження в файл (без графічного вікна)
- **1** — збереження в файл + показ в графічному вікні

**Залежності автоматично встановлюються при першому запуску!**
